RouteDocumentData

class RouteDocumentData : NSObject, NSSecureCoding

class for handling the selection of data to be stored in UIActivityShare Menus This wrapping functionality is required to that we can have all elements related to a specific route stored in a single encoded file ready to be shared throughout iOS
